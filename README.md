# 🌐 My Portfolio — Džemil Manić

A personal portfolio website built with **React + TypeScript + Vite**, showcasing my projects, skills, and ways to get in touch.

---

## 🚀 Live

> ⚠️ **Note:** Project demos may occasionally be offline as hosting is not always active.

---

## ✨ Features

- ⚡ Fast, responsive single-page application
- 🎨 Modern dark/light glassmorphism design with smooth animations
- 🖼️ Animated loading screen with progress bar
- 🗂️ Paginated project carousel (3 per page, responsive)
- 🏷️ Skills section grouped by category
- 📬 Contact section with all social links
- 📱 Fully responsive — mobile, tablet, desktop
- 🔔 Hosting status notice banner (easily toggled in code)

---

## 🛠️ Tech Stack

| Layer       | Technology                        |

## ⚙️ Running Locally

```bash
# Clone the repo
git clone https://github.com/dzemilmanic/Portfolio.git
cd Portfolio/portfolio

# Install dependencies
npm install

# Start dev server
npm run dev
```

App runs at `http://localhost:5173`
